### Mechanizmy Kontrolne

#### 1. Dynamic Staking APY
```
APY = Base_APY × (1 - TVL_Ratio)
gdzie TVL_Ratio = Total_Staked / Total_Supply
```

**Przykład:**
- Base APY: 50%
- TVL 10%: APY = 50% × 0.9 = 45%
- TVL 30%: APY = 50% × 0.7 = 35%
- TVL 50%: APY = 50% × 0.5 = 25%

#### 2. Daily Emission Cap
```
Daily_Cap = Total_Staking_Pool / Days_In_Period
Max_Daily = 171,232 $ASSASSIN (średnio)
```

#### 3. Monthly Quest Cap
```
Monthly_Quest_Rewards = Base_Cap × Activity_Multiplier
Max_Monthly = 6.67M $ASSASSIN (średnio)
```

## ⚠️ Ryzyka i Mitigacje

### Ryzyko: Nadmierna Inflacja
**Symptomy:**
- Cena spada > 50% w miesiąc
- TVL staking < 10%
- Burn rate < 10% emisji

**Mitigacje:**
1. Obniż Base APY (np. 50% → 30%)
2. Zwiększ token sinki (np. wyższe koszty craft)
3. Wprowadź nowe utility (więcej use cases)
4. Buyback & burn z treasury

### Ryzyko: Zbyt Niska Płynność
**Symptomy:**
- Slippage > 5% dla 10k USD trade
- Daily volume < 10k USD
- Spread > 2%

**Mitigacje:**
1. Incentive LP (farming rewards)
2. Partnership z MM (market makerem)
3. Listingi na CEX
4. Protocol-owned liquidity

### Ryzyko: Whale Manipulation
**Symptomy:**
- Top holder > 10% supply
- Duże sell-offy (>1% supply)
- Price manipulation

**Mitigacje:**
1. Vesting dla wszystkich alokacji
2. Anti-whale mechanizmy (max buy/sell)
3. Transparent tokenomics
4. Community monitoring

### Ryzyko: Low Retention → Sell Pressure
**Symptomy:**
- Churn rate > 70% M1
- Wszyscy gracze dumpują rewards
- Zero token sinki usage

**Mitigacje:**
1. Improve gameplay (retention)
2. Lock rewards (vest claim)
3. Increase utility (more use cases)
4. Better onboarding
